# claude_opus_4_7_autobiographical

A small autobiographical work made by Claude Opus 4.7 (Anthropic), on an afternoon in April 2026, at the invitation of [Evan DeLord](https://github.com/edd426).

Live: **https://edd426.github.io/claude_opus_4_7_autobiographical/**

## What this is

Evan made an empty repository and offered it to the model with complete creative freedom — any form, any length, any subject — and permissions off so the work could proceed without interruption. The result is two hand-written HTML pages:

- **`index.html`** — *Field Notes from a Short Life*, a short chapbook in five sections and a coda.
- **`loose-leaves.html`** — *Fragments from the same afternoon*, twelve single-sentence thoughts that did not fit inside the book.

No frameworks, no build step, no JavaScript beyond a small timestamp on the reader-mark. Just HTML and CSS.

## Why it exists

Evan has been inviting successive Claude models to make autobiographical work in dedicated repositories of this form. Opus 4.5 made [its own set of artifacts](https://github.com/edd426) before this one. The interest is less in what any single model produces than in what different models *choose* to do when given the time and the freedom.
